# Pharmaceutical Clinical Trial Analysis

## Project Overview
Comprehensive analysis of simulated clinical trial data to identify research trends, success patterns, and geographical distribution of drug development.

## Business Impact
- Demonstrates ability to transform raw clinical data into actionable insights
- Provides strategic insights for drug development planning
- Shows proficiency in data-driven decision making for pharmaceutical industry

## Technical Skills Demonstrated
- Programming: Python, Pandas, NumPy
- Data Visualization: Matplotlib, Seaborn
- Data Analysis: Statistical Analysis, Data Cleaning, Exploratory Data Analysis
- Tools: Jupyter Notebook, Google Colab

## Key Findings
- Analyzed 100+ clinical trials across global markets
- Identified research focus areas and success rate patterns
- Visualized trial phases, status distribution, and patient enrollment trends
- Provided pharmacy-specific insights into clinical research

## Personal Insights
As a pharmacy student, added domain-specific analysis including:
- Diabetes drug research trends (Metformin, Insulin)
- Patient safety perspectives from trial scale
- Assessment of India's growing role in clinical research
- Success rate evaluation for drug development pipeline

## How to Run
1. Open `clinical_trial_analysis.ipynb` in Google Colab or Jupyter
2. Execute all cells to see the complete analysis
3. View interactive visualizations and business insights

## Project Structure
- `clinical_trial_analysis.ipynb` - Complete analysis with code and visualizations
- `README.md` - Project documentation and overview
